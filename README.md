# Safta E-Commerce Backend

A secure and scalable E-Commerce Backend application developed using Java, Spring Boot, Hibernate, and MySQL. This project provides REST APIs for user authentication, product management, cart management, and order processing.

---

## Features

- User Registration & Login
- JWT Authentication & Authorization
- Product Management
- Cart Management
- Order Processing
- RESTful APIs
- MySQL Database Integration
- Role-Based Access Control
- Secure Backend Architecture

---
